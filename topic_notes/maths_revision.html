Fractions

Numerator = number on top
Denominator = number on bottom

Adding Fractions

Convert denominators (multiply top and bottom by same number) so they're the same.
Add the numerators and keep the denominator the same.

1/3 + 1/6
= 2/6 + 1/6
= 3/6

Subtracting Fractions

Same as above but subtract instead of add.

Multiplying Fractions

Multiple top numbers, and multiply bottom numbers.

1/5 x 2/3
= (1x2)/(5x3)
= 2/15

Dividing Fractions

Flip the 2nd fraction and multiply.

(1/2) / (1/6)
= (1/2) x (6/1)
= 6/2
= 3

Standard Form

4.3 x 10^4
= 43000.0
Move the decimal place to the right

4.3 x 10^-4
= 0.00043
Move the decimal place to the left.

Surds

SQRT(5)^2
= 5

2x(SQRT(5)) + 3x(SQRT(5))
= 5x(SQRT(5))

2x(SQRT(5)) x 3x(SQRT(5))
= SQRT(2^2 x 5) x SQRT(3^2 x 5)
= SQRT(4 x 5) x SQRT(9 x 5)
= SQRT(20) x SQRT(45)
= SQRT(20 x 25)
= SQRT(900)
= SQRT(100 x 9)
= SQRT(100) x SQRT(9)
= 10 x 3
= 30

Fractional Indices

5^2 = 25

25^(1/2) = SQRT(25) = 5

2^3 = 2 x 2 x 2 = 8

8^(1/3) = CUBEROOT(8) = 2

3^(1/2) x 3^(1/2) = 3^1 = 3

25^(3/2) = (25^(1/2))^3 = SQRT(25)^3 = 5^3 = 5 x 5 x 5 = 125

Algebra

2(y + 4) = 2y + 8

3y + 12 = 3(y + 4)

(y + 3 )(y + 4)
= y^2 + 4y + 3y + 12
= y^2 + 7y + 12

y^2 + 6y + 8
= (y + 2)(y + 4)

y^3 x y^7 = y^10 (multipling terms with indices involves adding thr indices together)

y^6 / y^2 = y^4 (dividing terms with indices involves subtracting their indices)

y = ax + c (a is the gradient, c is the intercept on the y axis)

Quadratic Equations

y^2 + 3y - 4 = 0
=> (y + 4)(y - 1) = 0
=> y + 4 = 0 and y - 1 = 0
=> y = -4, and y = 1

x^2 + 6x + 7
= (x + 3)^2 - 2
...then you can re-arrange this to find the value of x (and then the value of y) where the quadratic line on the plot has zero gradient (turning point) - this is useful in machine learning.
e.g. (x + 3)^2 - 2 = 0
=> (x + 3)^2 = 2
=> x + 3 = SQRT(2)
=> x = SQRT(2) - 3

Trigonometry

To find the length of a side of a right angled triangle, where c is the longest side:
a^2 + b^2 = c^2
This is useful for fiding the distance between points on a graph which is useful in machine learning.

The area of a triangle is:
(width x height) / 2
This is useful in statistics (and therefore machine learning) when trying to find a probability via calculating the area under a probability distribution graph.

Vectors

(1, 2) + (3, 4) = (4, 6)
(1, 2) - (3, 4) = (-2, -2)
2 x (1, 2) = (2, 4)
(2, 4) / 2 = (1, 2)

Factorials

n! = n(n-1)(n-2)...(1)
4! = 4 x 3 x 2 x 1
5! = 5 x 4 x 3 x 2 x 1
0! = 1
1! = 1

The number of ways to arrnage n items is n!
e.g. the number of ways to arrange 3 items is 3! = 3 x 2 x 1 = 6.
