Maths Revision

Notes on key GCSE Maths topics I've had to revise in order to learn the necessary statistics for Data Science.

Fractions

Numerator = number on top
Denominator = number on bottom

Adding Fractions

Convert denominators (multiply top and bottom by same number) so they're the same.
Add the numerators and keep the denominator the same.

1/3 + 1/6
= 2/6 + 1/6
= 3/6

Subtracting Fractions

Same as above but subtract instead of add.

Multiplying Fractions

Multiple top numbers, and multiply bottom numbers.

1/5 x 2/3
= (1x2)/(5x3)
= 2/15

Dividing Fractions

Flip the 2nd fraction and multiply.

(1/2) / (1/6)
= (1/2) x (6/1)
= 6/2
= 3

Standard Form

4.3 x 10^4
= 43000.0
Move the decimal place to the right

4.3 x 10^-4
= 0.00043
Move the decimal place to the left.

Surds

SQRT(5)^2
= 5
